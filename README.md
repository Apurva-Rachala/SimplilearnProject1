# SimplilearnProject1
As a Full Stack Developer, you have to build template using which, someone can purchase the same and use it. Background of the problem statement: As you have worked on web technologies in your previous company, you are asked to create a template which can be used by the consultancies or individual web developers to sell their templates. You must use the following: HTML/CSS: To create design of the template. Following requirements should be met: A few of the source code should be tracked on GitHub repositories. You need to document the tracked files that are ignored during the final push to the GitHub repository. The submission of your GitHub repository link is mandatory. In order to track your task, you need to share the link of the repository in the document. The step-by-step process involved in completing this task should be documented.
